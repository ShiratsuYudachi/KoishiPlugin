# koishi-plugin-chatluna-vector-store-management

[![npm](https://img.shields.io/npm/v/koishi-plugin-chatluna-vector-store-management?style=flat-square)](https://www.npmjs.com/package/koishi-plugin-chatluna-vector-store-management)


## 开发计划

1. 创建插件并完成依赖配置，让插件可以安装到 Koishi 并提供管理控制台示例页面 (10%)
2. 读取 `chatluna-vector-database-service` 的向量数据库配置（或者是手动填写），使用向量数据库对应的 Node.JS Adapter 访问到数据库 (20%)
3. 将向量数据库的内容展示到管理控制台页面内 (50%)
4. 使用向量数据库对应的 Node.JS Adapter 提供数据选择、添加、修改、删除功能 (80%)
5. 提供批量导入导出功能 (90%)
5. 为插件内的一些配置选项等提供说明文档，交付啦 (100%)
